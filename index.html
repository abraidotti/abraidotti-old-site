Minit

If you're new to learning front-end development, here's a handy script to scaffold a simple html-css-js site. I called it minit, short for micro init.

First, open a new file in a text editor and paste in these two lines:

#!/bin/bash
#Minit is a tiny script for tiny sites.

That first line starts with a shebang - #! - and this tells the system to follow the subsequent command. I'm using Bash, but if your shell is different, you'll want to update this line. I also recommend you add a comment that briefly describes the project, which I've done there.


Next, we'll greet the user with the $USER variable -- this reads in the system's current username.

We'll follow this up with a request for input. We can print to the screen by using the echo command:

echo "Hello, $USER!"
echo "Please give me a name for the init directory:"

read dirName


Now we'll want to check if the input is blank and/or if the directory is already present:

#Check if input is blank
if [ -z "$dirName" ]
then
echo "No argument supplied"
exit
fi

if [ -d "$dirName" ]
then
echo "The directory already exists!"
read answer
if [ $answer != 'y' ]
then
exit
fi


Take note that every loop ends in an "fi." This tells the system to end the if loop (and is analagous to "endif.")


Now we'll want to build the directory structure and confirm its location:

else
echo "Initializing $dirName at $PWD"
mkdir $dirName && cd $dirName && touch index.html
mkdir -p $PWD/css && touch css/styles.css && mkdir -p $PWD/js && touch js/main.js


We'll add some boilerplate code to index.html:

echo "<html>
<head>
<title>$dirName</title>
<meta charset="utf-8">
<link rel='stylesheet' type='text/css' href='./css/styles.css'>
<script type='text/javascript' src='./js/main.js'></script>
</head>

<body>
<h1>Hello, World!</h1>
</body>
</html>" > $PWD/index.html


And we'll make sure main.js works:

echo "document.addEventListener('DOMContentLoaded', function(){
console.log('main.js is hooked up!');
});" > $PWD/js/main.js

echo "$dirName scaffold initialized."

fi

And there's your first Unix script! We're not done yet, though.


Your next step is to navigate in your finder to the file, right-click on it, and hit "get info."

You'll see a "sharing and permissions" tab at the bottom. Hit the padlock at the bottom to unlock it. You'll need to provide your password as well. Once you unlock it, this file will be able to read and write.

Next, you'll need to save the file with a .sh extension, and then type the following command at your Bash prompt:

chmod 700 minit.sh

This tells the system your script is executable.

To run the file, paste it into your Sites directory and type

./minit.sh

Build a short sequence of inherited objects.

var Rock = function(size, color, hardness){
this.size = size;
this.color = color;
this.hardness = hardness;
}

var Gemstone = function(size, color, hardness, lustre, crystalSize){
Rock.apply(this, arguments);
this.lustre = lustre;
this.crystalSize = crystalSize;
}

var quartz = new Rock('large', 'rose', 7);
var diamond = new Gemstone('large', 'clear', 10, 'brilliant', 'small');
